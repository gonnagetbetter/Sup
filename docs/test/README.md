# Тестування працездатності системи

Для тестування використовувалася програма [Postman](https://www.postman.com/).

### Запуск cервера

Для запуску сервера необхідно перейти до директорії `/src/js/`:

```bash
cd ./src/js/
```

Та виконати 

```bash
node app.js
```
